# CuramFSCalendar



## Requirements

iOS 11.0

## Installation

CuramFSCalendar is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'CuramFSCalendar'
```
